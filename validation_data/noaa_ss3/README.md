# NOAA Stock Synthesis validation fixtures

These files are pinned, curated software-validation excerpts derived from the
official `nmfs-ost/ss3-test-models` repository. They are included to test
parsing and selected deterministic equations, and are not byte-identical copies
of the complete NOAA model folder. They are not fishery data sources and are
not evidence about the status of any stock.

The Omega downloader can retrieve complete current model folders from NOAA on a
computer with internet access. The embedded `Simple` fixture keeps automated
offline validation reproducible.
