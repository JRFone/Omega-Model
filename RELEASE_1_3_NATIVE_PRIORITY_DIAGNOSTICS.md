# Omega FISH Model 1.3.0

## Main changes

- C++17 production-model numerical engine.
- OpenMP candidate scoring.
- Forward-mode automatic derivatives for the native production objective.
- Native-gradient L-BFGS-B refinement.
- Native/Python numerical parity tests.
- Fully refitted likelihood profiles and two-dimensional profile surfaces.
- Genuine age-structured ASPM, ASPM-R, no-index and index-influence variants.
- Formal known-truth interval-coverage testing.
- Native Engine and Priority Diagnostics desktop workspace.
- Cross-platform native build and validation workflow.
- Packaged native-library support in the Windows desktop build.

## Evidence boundary

Release 1.3 improves numerical architecture and implements the three priority diagnostic systems. It does not establish complete Stock Synthesis equivalence, activate every optional C++ solver library, or prove superiority over SS3/ADMB/TMB across real assessments.
