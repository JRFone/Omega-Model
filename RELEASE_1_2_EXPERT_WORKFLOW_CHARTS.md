# Omega FISH Model 1.2.0

## Main additions

- Automatic and exploration expert-workflow modes.
- Complete diagnostic-gate interface.
- Finite-difference gradient checks.
- Jitter and independent optimizer agreement.
- Likelihood profiles and component-preference profiles.
- Retrospective, hindcast and MASE diagnostics.
- ASPM-style broad-trend diagnostic.
- Data-removal and weighting influence analysis.
- Optional age/length composition reweighting.
- Known-truth recovery and interval-coverage testing.
- Structural model ensembles and closed-loop MSE.
- Interactive Chart Studio.
- Saved personal chart profiles.
- WebGL rendering and display-only downsampling.
- Offline tabbed diagnostic dashboards.
- CLI commands for expert workflow and chart generation.
- GitHub Actions smoke tests for the new workspaces.

## Validation

The cumulative source suite contains 48 automated tests at release assembly. The final test report in the package records the exact run used to build the update.

## Claim boundary

The release improves integration, transparency and diagnostic automation. It does not yet prove that Omega is numerically superior to Stock Synthesis across the NOAA test-model catalogue.
